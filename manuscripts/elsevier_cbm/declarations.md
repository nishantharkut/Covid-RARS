# Declarations Checklist for CBM Submission

## Competing Interests

Draft statement: The authors declare no competing interests.

Author action before submission: every author must confirm this statement.

## Funding

Submission blocker: the corresponding author must confirm whether the work received any specific grant or institutional funding. If no specific funding supported the work, use:

This research received no specific grant from any funding agency in the public, commercial, or not-for-profit sectors.

## Ethics Approval and Consent to Participate

Submission blocker: the corresponding author must confirm the institutional ethics position for retrospective analysis of public/crowdsourced respiratory-audio datasets. Suggested wording only if institutionally accurate:

This study analyzed previously collected public/crowdsourced respiratory-audio datasets and did not involve prospective participant recruitment by the authors.

## Data and Code Availability

Draft statement: Public source datasets should be accessed through the original Coswara and COUGHVID maintainers under their respective terms. Derived aggregate tables, manuscript figures, and analysis code should be released in the project repository or an archival repository at submission or publication, subject to dataset redistribution constraints. COUGHVID should be described only as a cough-only external-transfer target.

## Author Contributions

Submission blocker: complete a CRediT statement using author initials before submission.
