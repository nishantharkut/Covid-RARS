# Manuscripts

This directory contains venue-specific manuscript drafts, compiled PDFs, shared figures, and source artifacts.

## Main Layout

| Path | Contents |
|---|---|
| `common_figures/` | Shared paper figures in SVG/PDF form |
| `source_artifacts/` | Data extracts used while writing manuscripts |
| `style_studies/` | Venue style notes and rebuild audits |
| `ieee_jbhi/` | IEEE JBHI draft files |
| `elsevier_eswa/` | Elsevier ESWA draft files |
| `elsevier_cbm/` | Elsevier CBM draft files |
| `npj_digital_medicine/` | npj Digital Medicine draft files |
| `_render_check/` | Rendered page checks |

Before external submission or public release, verify venue formatting, author metadata, generated auxiliary files, and third-party figure/data redistribution rights.
