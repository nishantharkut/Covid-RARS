# IEEE/JBHI Author Checklist

- [ ] Confirm author names, affiliations, ORCID identifiers, and corresponding author details.
- [ ] Confirm that the manuscript follows IEEEtran journal formatting.
- [ ] Confirm that all figures are original study figures and have sufficient resolution.
- [ ] Confirm that no clinical deployment, diagnostic-use, or state-of-the-art claim is made.
- [ ] Confirm that COUGHVID is described only as cough-only external transfer.
- [ ] Confirm that Coswara is described as the source-domain multimodal and temporal dataset.
- [ ] Confirm that all numerical results match the reproducible analysis outputs.
- [ ] Confirm that metadata-only results are framed as confounding controls, not proposed screening models.
- [ ] Add confidence intervals or explain their absence where required by reviewers.
- [ ] Add ethics, data-use, funding, competing-interest, and author-contribution statements.
- [ ] Confirm that code and derived result tables are ready for repository release.
- [ ] Run a clean LaTeX build and inspect the generated PDF before submission.
