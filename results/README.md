# Results

This directory keeps frozen result folders out of the repository root while preserving their original names.

## Subdirectories

| Path | Contents |
|---|---|
| `frozen/` | Final BTP outputs, corrected no-leakage Coswara outputs, CNN cough outputs, phase-3 outputs, and external-validation outputs |
| `representations/` | OpenSMILE, BEATs, and PANNs representation result bundles |

These folders are evidence artifacts. Do not overwrite them casually. New runs should write to a new dated folder or to the active package output paths first.
